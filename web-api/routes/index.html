Routes

Fetches informations about Difys

URL /difys-info
Method GET
Authentication true

Example data

{ "name": "difys", "version": "1.0.0" }

Returns plugins config from the array of plugins name

URL /plugins/config
Method POST
Authentication true

Example payload

["Map", "Shop", "Character"]

Example data

{
    "Map": { ... },
    "Shop": { ... },
    "Character": { ... }
}

Updates plugins config from the config object list

URL /plugins/config
Method PUT
Authentication true

Example payload

{
    "Map": { ... },
    "Shop": { ... },
    "Character": { ... },

    ...
}

Example data

empty
